# Verificación OTP

## Resumen

Cuando el usuario final quiere acceder al detalle completo de su ticket, el sistema envía un código OTP de un solo uso a su email.

Características del OTP:
- Tiene tiempo de expiración
- Tiene límite de intentos fallidos
- Es de un solo uso

> Flujo completo, manejo de errores y casos edge se documentan en D3.
